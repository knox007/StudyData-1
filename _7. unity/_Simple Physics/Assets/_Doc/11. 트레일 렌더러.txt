---------------------------------
11.	트레일 렌더러
---------------------------------

	1.	물체가 이동할때 잔상처럼 보이는 효과.

		-	검을 휘두를때 표현되는 궤적 이나 총알이 날아가는 궤적 등.
	

	2.	표현 방식.

		-	물체가 이동함과 동시에 동적으로 메쉬를 만들고

			일정시간 경과 후 생성된 메쉬를 제거해가는 방법.

		-	유니티에서는 Trail Renderer이라는 컴포넌트 제공.

		-	오브젝트 선택		->	Component	->	Effects	->	Trail Renderer

			또는

			오브젝트 선택		->	인스펙터 Add Component	->	Trail Renderer 검색후 체크
	
	
	3.	주요 속성

		-	Time
			
			-	메쉬의 지속시간

			-	값이 크면 궤적의 길이가 길어진다.


		-	Min Vertex Distance

			-	최소 버텍스간 거리.

			-	이 값이 크면 트레일렌더러 메쉬에 적용되는 폴리곤의 갯수가 줄어든다.

				-	물체가 직선 이동만 한다면 값을 크게하여 폴리곤의 갯수를 줄인다.

				-	포물선 이동을 한다면 값을 작게해서 부드러운 궤적을 연출한다.


		-	Width

			-	메쉬의 최대 폭.

			-	기본값은 1

			-	Curved Editor을 통해 앞부분과 끝부분의 폭을 조절 가능.

				-	에디터 상에서 마우스를 더블 클릭하면 빨간색 마름모형태의 기즈모가 나타남.

				-	이 기즈모를 드래그해서 폭과 커브의 기울기를 조절한다.

		



	4.	프로젝트 적용

		1)	Bullet 프리팹에 트레일 렌더러 적용.

		2)	트레일 렌더러에 적용할 매터리얼 설정.

		3)	메터리얼에 Trail.png 이미지 적용

		4)	메터리얼의 셰이더는 Particles / Addictive로 변경.

		5)	궤적을 밝게 표현하기위해 Tint Color 속성을 흰색으로 설정.
