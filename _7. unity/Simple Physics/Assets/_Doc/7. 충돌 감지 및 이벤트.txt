---------------------------------
1.	충돌 감지 조건
---------------------------------

	1.	두개의 오브젝트가 충돌을 감지하려면 다음의 조건이 만족되어야 함.

		-	충돌을 일으키는 두개의 게임오브젝트 모두 Collider 가 적용.

		-	움직이는 오브젝트에는 반드시 Rigidbody가 적용.

		|				|					|				|

		|	Rigidbody	|	------>			|				|
