---------------------------------
1.	주요 속성
---------------------------------

	1.	Main 모듈
	
		-	Duration	:	파티클 시스템이 작동하는 시간.

		-	Looping		:	반복여부 설정.

							Duration으로 설정한 시간이 종료되는 시점에 다시 처음부터 동작.

		-	Prewarm		:	한번 동작을 완료한 상태부터 loop재생 시작

							주로 반복되는 Loop 이펙트 작업시 사용.

		-	Start

				-	Delay	:	파티클 작동하기 전 대기 시간.

				-	Lifetime	:	파티클 수명

				-	Speed	:	초기 이동 속도.

				-	Size	:	초기 크기

		-	Gravity Modifier	:	중력효과

									0	:	무중력

									음수 :	역중력.
		-	Simulation Space

				-	Local	:	자신 또는 부모 오브젝트 이동시 파티클이 따라 움직임.

				-	World	:											따라 움직이지 않음.


		-	Play On Awake	:	
					
					체크		:	게임 재생 시점부터 곧바로 파티클 시스템 작동.

					체크 해제	:	스크립트 등에서 실행해줘야 함.

		-	Max Particles	:	파티클 시스템이 동작하는 동안 방출할 파티클 최대 수.

	
	
	2.	Emission 모듈	:	파티클 방출량 설정.

		-	Rate over Time	:	초당 파티클 방출량

						
	3.	Shape	모듈	:	파티클 방출 형태.

		-	Cone, Sphere, Box 형태로 설정 가능.

		-	방출 범위, 방출 각도, 방출 방향등 설정 가능.

	
	4.	Velocity over Lifetime

		-	파티클의 속도를 변경하는 힘 설정.

	
	5.	Color over Lifetime	/ Speed

		-	시간/속도 경과에 따른 색 변화 설정.

	6.	Size over Lifetime	/ Speed

		-	시간/속도 경과에 따른 크기 변화 설정.

	7.	External Forces

		-	Wind Zone의 영향을 받게 하는 설정.

	8.	Collision

		-	충돌시 일어나는 효과 설정

			-	바운싱, 소멸 등.

	9.	Texture Sheet Animation

		-	텍스쳐 애니메이션 형식 지정.

	10.	Renderer

		-	파티클 형태, 매터리얼등 설정.
