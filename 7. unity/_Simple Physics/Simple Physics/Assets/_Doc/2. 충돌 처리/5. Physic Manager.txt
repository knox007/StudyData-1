---------------------------------
1.	물리엔진 속성 설정
---------------------------------

	1.	Physics Manager

		-	Edit	-	Project Settings	-	Physics 선택.

	2.	중요 속성

		-	Gravity

			-	리지드 바디의 Use Gravity 체크시 적용되는 중력의 힘과 방향.

			-	기본값 Y축 -9.8

				-	중력 가속도 9.8 m/s^2

		-	Default Material

			-	두 물체가 충돌시 반작용에 대한 속성.

				None 설정시 각 리지드바디에서 개별적으로 설정가능.

		-	Sleep Threshold

			-	전역 에너지 임계값.

				리지드 바디의 운동에너지를 질량으로 나눈 값이 이 값보다 작으면

				리지드 바디가 휴면모드로 전환됨.( 리지드 바디 슬리핑 )
			
			-	리지드 바디 슬리핑은 유니티에서 자동으로 처리.

			-	휴면 모드에서 벗어나는 방법.

				1)	다른 리지드 바디 게임 오브젝트(컬라이더 포함)와 충돌

				2)	리지드 바디 속성 변경

				3)	AddForce등의 함수로 힘을 가할때.

		-	Queries Hit Triggers

			-	Raycast등의 컬라이더 충돌 체크기능 작동 여부설정.

		-	Layer Collision Matrix

			-	레이어 간의 충돌 감지여부를 선택적으로 설정.

			-	리지드 바디를 포함한 게임 오브젝트가 특정 레이어에서만 물리엔진의 

				영향을 받을수 있도록 하는 옵션.
