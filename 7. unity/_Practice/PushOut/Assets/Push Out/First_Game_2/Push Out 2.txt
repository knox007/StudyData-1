3.	캡슐의 이동 기능을 적용한뒤 전후 좌우 무작위로 이동해본다.

	물리적인 영향을 받도록 리지드 바디를 적용했기 때문에

	일정시간 이동하면 캡슐이 쓰러진다.

	이를 막기 위해 RigidBody의 Constraints를 수정한다.

	
	-	Constraints 역할.

		-	외부의 물리적인 영향이 발생하더라도 해당 부분에 대해서는 

			물리적인 작용을 제어하는 기능.

			-	Freeze Position	: 이동 제어.
			-	Freeze Rotation : 회전 제어.



4.	현재는 캡슐이 이동방향으로 회전하지 않는다.

	캡슐의 방향을 확인하도록 박스를 설치하여 캡슐의 차일드로 만든뒤 확인한다.

	박스의 트랜스폼
	-	위치 : 0, 0.5, 0.5
		스케일 : 0.3, 0.3, 0.3
