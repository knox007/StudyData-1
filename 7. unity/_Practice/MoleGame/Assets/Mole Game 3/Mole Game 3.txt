5.	상태머신의 상태변경 및 애니메이션 과정 흐름.

	-	Update 함수에서 구현.

		if(open상태)
			Open애니 -> open애니 종료 -> Idle상태

		if(Idle상태)
			Idle애니 -> Idle애니 종료 -> Close상태

		if(Close상태)
			Close애니 -> Close애니 종료 -> None상태

		if(Catch상태)
			Catch애니 -> Catch애니 종료 -> None상태


		->	상태를 시작하는 부분 / 상태를 진행하는 부분.

		->	애니메이션_On	:	상태 시작.
			애니메이션_Ing	:	상태 진행.
