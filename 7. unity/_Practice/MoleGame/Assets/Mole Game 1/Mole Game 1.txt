1.  두더지 구멍 상태머신 만들기

    * 상태머신 설계
    
      - None State
        - 기본 상태, 빈구멍.
        
      - Open
        - 두더지 등장, 두더지가 등장하는 애니.
        
      - Idle
        - 두더지 대기, 대기 중인 애니.
        
      - Catch
        - 터치 발생, 두더지 피격 애니.
        
      - Close
        - 두더지 퇴장. 두더지가 사라지는 애니.
        
    * 플로우.
    
      None -> Open -> Idle -> Close ----> None
              |         |             |
              |         |             |
              --------------> Catch ---
