-------------------------
1.	반복문이란?
-------------------------

	1.	필요성..

		-	동일한 작업을 일정 횟수( 또는 일정 조건 )동안 반복하고 싶을때 필요.

	2.	종류

		-	while

		-	do ~ while

		-	for




-------------------------
2.	while
-------------------------

	1.	형식

		while( 반복 조건 )
		{

			반복 구간
		}

	
		예)	
			int num = 0;
	
			while( num < 3 )
			{	
				printf("Hello !!\n");

				++num;
			}
				
			( "while" 참고 )
