/// C언어의 메모리 구조 /////////////////////////////////////////////

------------------
1.	메모리 구성
------------------

	-	프로그램 실행시 운영체제에 의해 할당되는 메모리 공간.

		-------------------------
		|	코드영역	|												
		-------------------------------------------------------------------------
		|	-	실행할 프로그램의 코드가 저장되는 메모리 공간.		|
		|									|
		|	-	CPU는 여기에 저장된 명령문들을 하나씩 가져가서 실행.	|
		|									|
		|	-	변수가 할당되는 영역은 아님.				|
		-------------------------------------------------------------------------
		|	데이터 영역	|
		-------------------------------------------------------------------------
		|	-	전역변수, static 변수.					|
		|									|
		|	-	프로그램 시작시 메모리 공간에 할당.			|
		|									|
		|	-	프로그램 종료시 메모리 공간에서 소멸.			|
		-------------------------------------------------------------------------
		|	힙 영역		|
		-------------------------------------------------------------------------
		|	-	동적으로 생성되는 변수.					|
		|									|
		|	-	프로그래머가 원하는 시점에 변수의 할당과 소멸이 이뤄짐.	|
		-------------------------------------------------------------------------
		|	스택 영역	|
		-------------------------------------------------------------------------
		|	-	지역 변수, 매개 변수					|	
		|									|
		|	-	함수(또는 코드블록)안에서 생성되고 경계를 벗어나면 소멸.|
		-------------------------------------------------------------------------


---------------------------------------------
2.	프로그램 실행에 따른 메모리의 상태 변화.
---------------------------------------------

	예)
			int gSum = 25;

			int main()
			{
				int num1 = 10;
				fct(num1);
				num1++;
				fct(num1);

				return 0;
			}

			void fct( int n )
			{
				int num2 =10;

				......
			}
			
			1)	프로그램의 시작은 main 함수의 호출에서부터!!

				-	실제로는 데이터 영역이 먼저 초기화.

				-	전역변수, static 변수가 먼저 데이터 영역에 할당된 후 main 함수 호출.


			2)	main함수나 fct함수의 지역변수나 매개변수가 스택 영역에서 할당과 소멸 반복.

			3)	return 문 실행되면 프로그램 종료.

			4)	운영체제에 의해 할당된 메모리 공간 전체가 반환.

				-	전역변수 소멸도 이때 이뤄짐.
